# No Shame Webpage

No Shame is a storytelling and advocacy webpage focused on highlighting women’s health challenges, particularly menstrual health struggles faced by displaced Sudanese women during conflict and crisis.

The project combines visual storytelling, research-driven advocacy, and media design to create awareness and encourage action, donations, and community engagement.

## Pages

- `index.html` — Story of Ameera
- `help.html` — Organizations that help with the issue
- `register.html` — Volunteer registration form

## Purpose

This project aims to:
- Raise awareness about menstrual health inequity
- Humanize data through storytelling
- Support advocacy for women in underserved communities
- Encourage volunteering, donations, and public engagement
